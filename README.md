# 📝 Web Forms – Working with User Data

A modern and responsive User Registration Form built using **HTML5, CSS3, and JavaScript**.

This project demonstrates proper form structure, styling, and client-side validation while providing an attractive and user-friendly interface.


## 🚀 Features

- ✅ Modern Glassmorphism UI Design
- ✅ Responsive Layout
- ✅ HTML5 Input Types
- ✅ Client-Side Validation using JavaScript
- ✅ Custom Error Messages
- ✅ Success Message After Submission
- ✅ Smooth Input Focus Effects
- ✅ Clean and Professional Styling



## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox, Gradient, Transitions)  
- JavaScript (DOM Manipulation & Validation)  
- Google Fonts
